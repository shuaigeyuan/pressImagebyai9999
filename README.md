# 图片压缩网站

这是一个简单的图片压缩网站，允许用户上传图片并进行压缩。

## 功能特点
- 支持 PNG、JPG 格式图片上传
- 实时预览原图和压缩后的图片
- 显示压缩前后的文件大小
- 支持压缩后图片下载
- 苹果风格的现代化界面设计

## 项目结构
image-compressor/
├── index.html
├── css/
│ └── style.css
├── js/
│ └── main.js
└── README.md


## 使用方法
1. 创建项目文件夹 `image-compressor`
2. 按照上述结构创建对应的文件
3. 将代码复制到相应文件中
4. 在浏览器中打开 `index.html` 即可使用

这个项目实现了图片上传、压缩、预览和下载功能，并采用了苹果风格的现代化设计。你可以根据需要进一步调整样式和功能，以实现更好的用户体验和视觉效果。
